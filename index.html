Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://nixgates.github.io/packages/ - Name it Nixgates
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Nixgates", and install nixgates.repository.zip
